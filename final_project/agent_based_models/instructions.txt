##### START view: #####
Either go to the live version, or open the index.html file in either the LV or the LVlite directory to start a localhost


##### START Running avg.py: #####
From either the fullData or simplifiedData directory run:

python3 ../avg.py <option>

where option can be Pred, Prey, or Iters, for example:
python3 ../avg.py Iters




##### START Running plotMAS.py: #####
From either the fullData or simplifiedData directory run:

python3 ../plotMAS.py <option>

where option is the name of any .log file generated by the model, for example:
python3 ../plotMAS.py const9



##### START Collecting Data: #####
Run a local copy of the code, or go to the link provided
Open the console, and at any point save the contents of the console as a .log file.
